jQuery Practice

.hide/.show

Click the .hide button to hide this paragraph! Click the .show button to show this paragraph.

.toggle

Click the .toggle button to toggle between the .hide and .show functions!

.slideDown/.slideUp

Click the .slideDown button to slide the text down

.slideToggle

Click the .slideToggle button to display or hide matched elements with a sliding motion

.fadeIn/.fadeOut

Display the matched elements by fading them to opaque or hide the matched elements by fading them to transparent

.before/.after

Insert content specified by the parameters after/before each element in the set of matched elements

.append

Insert content, specified by the parameter, to the end of each element in the set of matched elements

.html

Get the HTML contents of the first element in the set of matched elements or set the HTML contents of every matched element.

.attr

Hover over the picture to change the size of the picture

.val/.text
